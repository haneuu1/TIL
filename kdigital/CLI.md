# CLI 

- CLI(Command Line Interface)

Terminal

- GUI(Graphic User Interface)

## 기초 명령어

| 명령어   | 기능                       |
| -------- | -------------------------- |
| `mkdir ` | 폴더 생성 (make directory) |
| `touch`  | 파일 생성                  |
| `cd`     | 폴더 이동                  |
| `mv`     | 파일 이동                  |
| `ls`     | 내용 확인                  |
| `rm`     | 파일 삭제                  |

`cd .` : 현재 위치로 (변화없음)

`cd ..` : 위로 가기 

`ls -a` : 모든 내용 보기 (숨겨진 내용 포함)

`rm -r` : 폴더 삭제
